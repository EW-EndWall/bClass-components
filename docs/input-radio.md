# Input radio

## Model 1

- .input-radio-m-1

### example

```html
<div class="w-10 m-x-auto m-y-2 p-1 border-0-1 border-radius-0-5">
  <label>
    <input
      type="radio"
      class="input-radio-m-1"
      name="fav_language"
      value="HTML"
      checked
    />
    <span>HTML</span>
  </label>
  <br />
  <label>
    <input
      type="radio"
      class="input-radio-m-1"
      name="fav_language"
      value="CSS"
    />
    <span>CSS</span>
  </label>
  <br />
  <label>
    <input
      type="radio"
      class="input-radio-m-1"
      name="fav_language"
      value="JavaScript"
    />
    <span>JavaScript</span>
  </label>
</div>
```
